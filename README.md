### RackNerd可以更换IP吗？

RackNerd可以在后台自助更换IP，具体规则如下：

购买VPS 72小时内第一次更换 IP 免费，免费更换IP后不会再获得任何免费IP更改机会。购买VPS 72小时后， 更换IP一次性收费3美金。
